FsmdaEqvChecker
Kunal Banerjee
Sep 25, 2014


Pre-requisites:
1. gcc 4.7.2
2. flex 2.5.35
3. bison 2.5


Please execute the following commands to install and run FsmdaEqvChecker.

/*********************************************************************************/

In the folder fsmda2dot:

make                                        <-- This will install FsmdaEqvChecker.

cd bin			

./FsmdaEqvChecker <filename1> <filename2>   <-- To run fsmda2dot.

/*********************************************************************************/


Some original and their corresponding transformed FSMDAs can be found in the 
"Benchmarks" directory of the original folder "EquivalenceChecker".



For more information (and on discovery of bugs), please contact:
kunal.banerjee.cse@gmail.com
